Plot.ly Cheat Sheet

Define a single trace 

var x_data = [1, 2, 3, 4];
var y_data = [10, 20, 30, 40];
var trace1 = {
				x: x_data,
				y: y_data,
				mode: 'markers'
			};
Styling a plot or trace 

var trace1 = {
				x: x_data,
				y: y_data,
				mode: 'markers',

				// Styling
				marker: {
					color: 'red',
					size: 20
			}
			};
Use the provided 'Letters' data set 

var trace1 = {
				// x coord data for letter 'a'
				x: data.a.x,

				// y coord data for letter 'a'
				y: data.a.y,
				mode: 'markers'
			};

// letter 'R'
data.r.x

// space
data.space.x

// apostrophe
data.apost.x

// question mark
data.quest.x

// exclamation mark
data.exclam.x
Make a new plot 

Plotly.newPlot(graph_location, [trace1], layout)
"graph_location" = place in page to put plot

"[trace1]" = Set of pre-defined data sets. Here only one such trace, "trace1"

"layout" = predifined set of parameters for general appearence of the plot
Define a function 

var my_function = function(my_arguments){my code;};
'my_arguments' are optional.

Used only when used in my_function's 'my code'.
Run code on a button click 

Plotly.d3.select('#my_button').on('click', function(){
	my button click code;
})
Animate a plot.ly plot (see eg below) 

Plotly.animate(graph_location, 
	{
		data: [new_trace1]
	},

	{
		timing parameters
	}
)
Example of Animating a plot.ly plot 

Plotly.animate(graph_location, 
	{
		data: [
			{
				x: new_x_data, 
				y: new_y_data 
			}
		]
	},
	{
		// Total time before next animation
		frame: {duration:2000},

		// time of actual animation
		transition: {duration: 1500}
	}
)
